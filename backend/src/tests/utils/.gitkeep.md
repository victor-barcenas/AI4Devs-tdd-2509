# Test Utilities

This directory contains helper functions and utilities used across different test types.

## Structure
- `database.utils.ts` - Database setup and cleanup utilities
- `testData.factory.ts` - Test data factory functions
- `assertion.helpers.ts` - Custom assertion helpers

## Usage Example
```typescript
import { createTestCandidate } from '../utils/testData.factory';
import { cleanupDatabase } from '../utils/database.utils';

const candidate = createTestCandidate({
  firstName: 'Test',
  email: 'test@example.com'
});
```

## Guidelines
- Create reusable utilities for common test operations
- Keep utilities focused and single-purpose
- Document utility functions thoroughly
